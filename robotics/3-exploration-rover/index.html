Exploration Rover — “LizzHard”

First-year electronics design project: a web-controlled rover that detects and decodes four signals from arena “lizards” — ultrasound (UART name), infrared (age via pulse frequency), radio (AM), and magnetic polarity — to classify each target. :contentReference[oaicite:1]{index=1}

See Gallery

Overview

Project Goals
  • Build a drivable base and iterate to a complete sensing rover.
  • Detect & decode Ultrasound, IR, Radio, Magnetics reliably.
  • Serve a simple control UI (Wi-Fi) to drive and request readings.
  • Classify lizards (name + species) and log results on the web page. :contentReference[oaicite:3]{index=3}
Team

Group 28: Benoit BM (manager), Jay Dong, Chenxuan Xi, John Shin, Akarsh Gopalam, Misha Ischenko. :contentReference[oaicite:4]{index=4}

Key Specs
MCU
Adafruit Metro M0 Express + Wi-Fi shield
Comms
Web UI (HTTP GET/POST), JSON responses
Sensors
Ultrasonic (40 kHz ASK→UART), IR (130–1000 Hz), AM Radio (≈89 kHz), Hall (polarity)
Analog Front-Ends
MCP6022 op-amps, envelope detectors, filters, comparators
Drive
Two-wheel DC, H-bridge PWM; auto-stop safety
Chassis
Light 3-wheel base + front servo’d antenna mounts

Sensor Modules

Each sensor was iterated from detection → amplification → robust decoding. :contentReference[oaicite:5]{index=5}

Ultrasonic (Name)
  • 40 kHz carrier, ASK; 600 baud UART (4 ASCII chars + “#”).
  • Transducer → gain stage → envelope → threshold → comparator → decoded in firmware (timed sampling). :contentReference[oaicite:6]{index=6}
Infrared (Age)
  • Frequency measured 130–1000 Hz; high-pass filter (~100 Hz) to remove ambient, amplified & thresholded. :contentReference[oaicite:7]{index=7}
Radio (AM)
  • Custom air-core solenoid (~435 µH) + parallel C tuned near signal, envelope detection, multi-stage amplification. :contentReference[oaicite:8]{index=8}
Magnetic (Polarity)
  • SS49E hall sensor; differential stage biased to amplify ΔV vs baseline → North/South classification. :contentReference[oaicite:9]{index=9}
